CSS 박스 모델

아래 두 박스는 모두 가로(width) 3000px / 세로(height) 100px로 부여했음.
BUT!!!

content-box
실제 가로폭
= width(300px) + padding(40px) + border(4px)
실제 세로폭 = height(100px) + padding(40px) + border(4px)
border-box
실제 가로폭 = width(300px)
실제 세로폭 = height(100px)

display 속성 제어

기본적으로 inline 태그(요소)는 가로/세로 크기를 변경할 수 없음
해당 태그에 display: inline-block을 적용하면 크기를 조절할 수 있음
