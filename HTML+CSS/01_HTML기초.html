p 태그: 줄바꿈, 복수 공백 무시

HTML은 기본적으로 코드 상의 줄바꿈과 여러 개의 공백을 무시한다.
줄을 바꾸고 싶다면 br 태그를 이용해야 한다!

공백을 여러 칸으로 표현하고자 한다면 특수기호(엔터티 코드)를 사용해야 한다.
    공백을 의미하는 코드는 &nbsp;이다.
수식 표현: n2 < n2;처럼 부등호는 특수 기호로 표현해야 안전하다.
자주 사용하는 코드: < (미만), > (초과), & (앰퍼샌드&)

pre 태그: 작성한 대로 표현

이 곳은 pre 태그 내부이다. 
작성한 내용 그대로, 줄바꿈과  공백이 화면에 출력된다.
        들여쓰기도 그대로 표현된다.

b, strong 태그: 글자를 굵게 표시하는 태그

b: 단순 볼드체
strong: 볼드체 표시 및 중요 내용 강조에 사용
  스크린 리더기가 강조하여 읽어주는 효과를 가진다.

i, em 태그: 글자를 기울여서 표시하는 태그

i: 단순 기울임체 em: 기울임체로 표시 및 특정 단어 강조에 사용

그 외의 서식 태그

u: 밑줄 표시 태그
s: 취소선 표시 태그
mark: 글자 배경을 노란색으로 표시하는 태그

sub: 아래첨자 표시 태그 (H2O)

sup: 위첨자 표시 태그(x2)
