Window Object

The top-level object in JavaScript that controls the browser window.
It includes not only the 'document', but also 'location', 'screen', 'navigator', 'history', etc.
자바스크립트의 최상위 객체로, 브라우저 창을 제어
document 외에도 location, screen, navigator, history 등의 객체를 포함

Timer Function (setTimeout / setInterval)

- setTimeout: Runs a function once after a specified delay
- setInterval: Repeatedly runs a function at specified time inervals
- setTimeout : 지정한 시간이 지난 후 함수를 1회 실행
- setInterval : 지정한 시간 간격마다 함수를 반복 실행

Open / Close Pop-up

window.open()

URL Manipulation: location
