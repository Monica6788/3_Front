JavaScript Event

Event: user action or change of status occurred on browser
(click, keydown, keyup, drag, submit, load)
Event Handler: A callback function when specific event occurs

[1] Classic Event Model

element.onclick = function

Only one eventHandler can be registered per one event. (Overwrited)
Can be removed by 'onclick = null'

Standard Event Model

element.addEventListener(event-name, handler)

Can register multiple event handlers for the same event.
A specific event handler can be removed by 'removeEventListener()' method.

Event Object

target / currentTarget

Prevent Default Actions

event.preventDefault
