JavaScript Elemental Objects(자바스크립트 기본 객체)

  • window: 브라우저 창마다 생성되는 최상위 객체
    (window. 함수 사용 시 window. 생략 가능)
    window: Top-level object that is created for each browser window/tab
    (The 'window.' can be omitted when calling its functions/methods)
  • document: window.document
    현재 브라우저에 표시된 DOM을 관리하는 객체
    Object that manages DOM on current browser

Data Input Methods

prompt("메시지"): 사용자에게 메시지와 함께 입력창을 가지고 있는 창을 표시
- [확인] 클릭 시: 입력된 값(문자열) 반환
- [취소] 클릭 시: null 반환

prompt("message"): Display a Dialog box with message and an input box to users
- If [OK] is clicked: return the input value(String)
- If [cancel] is clicked: return null
