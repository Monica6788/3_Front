JS 기초

웹의 3요소

  • HTML: Define the structure and content of a web Document (DOM Tree structure)
  • CSS: Define the style and layout of DOM elements
  • JavaScript: Dynamically manipulate the DOM when events occur

JavaScript?

웹 브라우저에서 주로 사용되는 인터프리터 방식의 스크립트 언어
크롬 등의 브라우저에서 V8과 같은 자바스크립트 엔진이 내장되어 있어 별도의 설치 없이 실행 가능
인터프리터 방식: 코드를 한 줄씩 읽어가며 즉시 실행하는 방식

Interpreted scripting language that is usually used on web browsers
Execution available without specific installation with embeded JavaScript Engine like V8 in browser like Chrome
Interpreter approach: Read and Execute code line by line

자바스크립트 적용 방식

  1. Inline: Directly write in 'event' attribute of HTML tag
  2. Internal: Write in 'script' tag of HTML document
  3. External: Write code in a separated .js file

Inline(인라인 방식)

<tag onEvent-name="JavaScriptCode">
<태그명 on이벤트명="자바스크립트 코드">

Internal

script 태그 내에 자바스크립트 코드 작성
HTML 요소를 조작할 경우, 스크립트가 대상 요소보다 아래에 위치해야 정상적으로 인식됨

External

Manage by creating a separated JavaScript file(.js)
